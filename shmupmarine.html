ShmupMarine

Shoot Them Up Game - C# & Godot

Project Overview

"ShmupMarine" is a thrilling shoot-them-up game that challenges players to defeat waves of enemies while navigating complex patterns. Created using C# and Godot, this project emphasizes enemy AI behavior and engaging gameplay mechanics.

Technical Details

Enemy AI

The enemy AI is designed to challenge players with varying attack patterns and behaviors, ensuring a unique gameplay experience.

  • Utilizes a State Machine to control enemy states.

Bullet Patterns

Players face dynamic and unique bullet patterns that changes for each enemy, keeping the gameplay engaging.

  • Implemented a system to randomize bullet trajectories for unpredictability.
  • Different bullet types with varying speeds and effects add depth to combat.

Boss Design

A unique boss fight at the end of the game to test the players skill.

  • The boss has a health bar and multiple attack phases, increasing difficulty over time.
  • Visual and audio cues signal boss attacks, enhancing player awareness.
