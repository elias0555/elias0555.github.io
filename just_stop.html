Just Stop!

One Button Game - C# & Godot

Project Overview

"Just Stop!" is a one-button multiplayer game where players must respond quickly to on-screen feedbacks. Developed using C# and Godot, the game fosters competition and quick reflexes.

Gameplay Video

Watch the gameplay of "Just Stop!" showcasing its one-button mechanics and thrilling multiplayer competition.

Technical Details

Local Multiplayer System

The game supports local multiplayer gameplay, allowing friends to compete in real-time.

  • Implemented player input detection to capture simultaneous button presses.
  • Score tracking for each player with dynamic UI updates.

Special Feature

Each round the players have a short period of time to react and use the special acceleration feature.

  • Used a random timer to add challenge and unpredictability
  • The turns in the game ensure that each player maintains a consistent speed to preserve the integrity of the game loop.

Scoreboard System

The scoreboard tracks player scores and displays results at the end of each round.

  • Implemented a leaderboard to showcase high scores over multiple rounds.
  • Real-time score updates maintain competitive spirit among players.

Juiciness

The game was designed from the ground up to be as responsive and engaging as possible.

  • Used Godot's particle system to empower every player action
  • A real work was made on the UI to make it clean, simple, juicy and responsive
