Horrorush

Puzzle Game Remake - C# & Unity

Project Overview

"Horrorush" is a remake of an original puzzle game made by Two Tribes. The originality behind this remake remains in the sound design and the atmosphere.

Gameplay Video

Experience the puzzle solving game "Horrorush" and its atmosphere in this gameplay showcase.

Technical Details

Cube Movement & Collision system

The game features a unique gameplay mechanic where a cube has its own behavior, which changes based on the type of tile it encounters.

  • Cube movement and rolling is writen with maths only and quaternion logic.
  • No built in collision detection is used. Only Maths with raycasts.

Android & IOS ready

The game was developped to handle mobile controls.

  • Used unity Input system to handle screen touches.
  • It added a real challenge to make the game Android or IOS ready.

Scalable UI

The button sections on the level UI was designed totaly by code.

  • Maked the level setup really easy with the use of scriptable objects.
  • Used the scriptable objects to instantiate the right button type on screen with the right values..

Asset Creation

All assets were created in-house to ensure a cohesive art style and experience throughout the game.

  • Utilized Blender and Procreate for 3D modeling and animation.
  • Textures and UI elements were designed with Photoshop.

Sound Design

All sounds were made and mixed in-house to ensure a cohesive sound design with the art style.

  • Utilized Audacity for sound mixing and making.
